# Task 1: Star Wars Characters - Data Fetching Practice

## Task Description
To solidify your understanding of React hooks and API integration, you are revisiting the Star Wars universe. Using a free API ("https://workintech-fe-ecommerce.onrender.com/characters"), you will fetch character data and display it dynamically on the screen, following the visual design ("https://i.ibb.co/tCDdkSf/s6d5-task1-design.png") provided.

**Requirements:**
- Data Fetching: Implement a fetch mechanism within App.jsx to retrieve Star Wars character information from the provided API.
- State Management: Store the character list in a state and handle the loading process.
- Component Rendering: Map through the data to render each character's details (name, height, mass, etc.) as shown in the reference image.
- Step-by-Step Logic: Follow the specific implementation steps outlined within the App.jsx file.

**Working files:** src/App.jsx
