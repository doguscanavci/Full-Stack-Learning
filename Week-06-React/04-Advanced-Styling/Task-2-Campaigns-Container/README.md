# Task 2: Day Project - Campaigns Container (Carousel)

## Task Description
To enhance the user experience of the shopping site, you decided to implement a campaign showcase area at the top of the page. Using the Carousel component from Reactstrap, you will display promotional offers dynamically.

**Requirements:**
- Carousel Integration: Implement the Carousel component from the Reactstrap library within Offer.jsx.
- Data Binding: Import sahteVeri and map the title and text properties to the appropriate Carousel labels and captions.
- Functionality: Ensure the Carousel is fully functional, allowing users to slide through different offers.
- Layout: Position the Offer component at the top of the application within App.jsx.

**Working files:** src/components/Offer.jsx, src/App.jsx

---

## My Learning Journey & Reflection
Library Integration: I practiced integrating third-party UI libraries like Reactstrap to quickly build complex components like a Carousel.
Dynamic Mapping: I improved my skills in mapping local data objects to specific component properties, ensuring the UI reflects the underlying data structure accurately.

# Görev 2: Gün Projesi - Kampanyalar Container (Carousel)

## Görev Tanımı
Alışveriş sitesini daha çekici hale getirmek için sayfanın en üstünde kampanyaların sergilendiği bir alan oluşturmaya karar verdin. Reactstrap kütüphanesinin Carousel bileşenini kullanarak promosyon tekliflerini dinamik bir şekilde göstereceksin.

**Beklenenler:**
- Carousel Entegrasyonu: Reactstrap kütüphanesinden Carousel bileşenini Offer.jsx içinde uygula.
- Veri Bağlama: sahteVeri dosyasını import et; title ve text özelliklerini Carousel üzerindeki doğru başlık ve açıklama alanlarına yerleştir.
- Fonksiyonellik: Carousel'in sorunsuz çalıştığından ve kullanıcıların teklifler arasında geçiş yapabildiğinden emin ol.
- Yerleşim: Offer bileşenini App.jsx içinde sayfanın en üst kısmına konumlandır.

**Çalışma dosyaları:** src/components/Offer.jsx, src/App.jsx

## Gelişim Süreci ve Notlarım

Dinamik Eşleştirme: Yerel veri objelerini bileşen özellikleriyle eşleştirme (mapping) becerimi geliştirerek, verinin arayüzde doğru noktada gösterilmesini sağladım.
