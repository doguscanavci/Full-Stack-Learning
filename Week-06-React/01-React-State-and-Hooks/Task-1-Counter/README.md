# Task: React Counter Component

## Task Description
Congratulations! You've officially joined the React Team. To kick off your training, your first task is to build a functional Counter component based on specific requirements. This task focuses on understanding how data changes over time in a React application using the useState hook.

**Requirements:**
- Implement a counter that can increase, decrease, and potentially reset.
- Use the useState hook to manage the counter's value.

**Tip:** 
- Determine how many states you need to track the value.
- Use the syntax const [state, setState] = useState(INITIAL_VALUE) to define your state.
- Utilize ternary operators for conditional rendering or dynamic styling (e.g., changing color if the value is negative).

**Working files:** Counter.jsx

---

## My Learning Journey & Reflection
Introduction to State: I learned that in React, "State" is the data that determines what is rendered on the screen. When the state updates, React automatically re-renders the component to reflect the changes.
Declarative UI: Unlike Vanilla JS where I manually manipulated DOM elements (e.g., textContent = count), I now declare how the UI should look based on the current state.

# Görev: React Sayaç Componenti

## Görev Tanımı
Tebrikler, artık resmen React Takımı'ndasın! Eğitimin ilk adımı olarak, senden istenen isterlere göre çalışan bir sayaç (counter) bileşeni oluşturman gerekiyor. Bu görev, bir React uygulamasında verilerin zamanla nasıl değiştiğini ve bu değişimin ekrana nasıl yansıdığını anlamanı sağlayan useState hook'una odaklanmaktadır.

**Beklenenler:**
- Artırma, azaltma ve (isteğe bağlı) sıfırlama özelliklerine sahip bir sayaç mekanizması kurmak.
- Sayacın değerini yönetmek için useState hook'unu kullanmak.

**İpucu:** 
- Sayacın değerini takip etmek için kaç tane state tanımlaman gerektiğini belirle.
- Yeni bir state tanımlamak için const [state, setState] = useState(BASLANGIC_DEGERI) yapısını kullan.
- Şartlı gösterimler veya dinamik stiller (örneğin sayı negatifse rengin değişmesi) için ternary operatör (üçleme) kullanabilirsin.

**Çalışma dosyaları:** Counter.jsx

## Gelişim Süreci ve Notlarım
State (Durum) Kavramı: React'te "State"in ekranda neyin görüneceğini belirleyen veri olduğunu öğrendim. State güncellendiğinde, React değişikliği yansıtmak için bileşeni otomatik olarak tekrar render eder.
Deklaratif Arayüz: Vanilla JS'de yaptığım manuel DOM manipülasyonlarının (örneğin textContent = count) aksine, artık arayüzün mevcut duruma göre nasıl görünmesi gerektiğini beyan ediyorum.
