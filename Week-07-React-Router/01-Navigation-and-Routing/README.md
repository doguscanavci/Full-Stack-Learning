### English

# Task 1: Day Project - Movies Page Navigation

## Task Description
You have been assigned to a team project to build a functional Movies Page. Your primary goal is to implement the core logic and navigation using React Router before the UI team applies the final styling. You are responsible for 7 specific sub-tasks distributed across the application.

**Requirements:**
- App.jsx (3 Tasks): Set up the main BrowserRouter, define the primary Routes, and manage the state for the "Saved Movies" list.
- Film.jsx (2 Tasks): Implement the link structure to navigate to individual movie details and ensure the movie data is correctly passed.
- FilmDetaylari.jsx (1 Task): Use the useParams hook to capture the movie ID from the URL and fetch/display the specific movie's data from sahteVeri.js.
- KaydedilenlerListesi.jsx (1 Task): Create the logic to display and navigate through the movies that the user has marked as "Saved".

**Working files:** src/App.jsx, src/components/Film.jsx, src/components/FilmDetaylari.jsx, src/components/KaydedilenlerListesi.jsx

---

## My Learning Journey & Reflection
Client-Side Routing: I learned how to create a seamless multi-page experience without refreshing the browser using Link and NavLink components.
Dynamic Routing: I practiced using URL parameters (:id) to render dynamic content based on the user's selection.

---

## Tech Stack
- React.js
- React Router Dom (v6+)
- JavaScript (ES6+)

### Türkçe

# Görev 1: Gün Projesi - Filmler Sayfası Navigasyonu

## Görev Tanımı
Takım olarak yeni bir işe atandınız ve senin görevin, React Router kullanarak uygulamanın çekirdek navigasyon mantığını kurmak. UI ekibi daha sonra senin kurduğun bu yapı üzerine styling uygulayacak. Uygulama genelinde 7 farklı alt görevi tamamlaman bekleniyor.

**Beklenenler:**
- App.jsx (3 Görev): Ana BrowserRouter yapısını kur, temel Routes tanımlamalarını yap ve "Kaydedilenler" listesi için gerekli state yönetimini hazırla.
- Film.jsx (2 Görev): Bireysel film detaylarına gitmek için link yapısını kur ve film verilerinin doğru şekilde iletildiğinden emin ol.
- FilmDetaylari.jsx (1 Görev): URL üzerinden film ID'sini yakalamak için useParams hook'unu kullan ve sahteVeri.js içinden ilgili filmi bulup ekrana bas.
- KaydedilenlerListesi.jsx (1 Görev): Kullanıcının kaydettiği filmleri listeleyen ve bu listeden filmlere geri dönülmesini sağlayan mantığı kur.

**Çalışma dosyaları:** src/App.jsx, src/components/Film.jsx, src/components/FilmDetaylari.jsx, src/components/KaydedilenlerListesi.jsx

## Gelişim Süreci ve Notlarım
Client-Side Routing: Sayfayı yenilemeden, Link ve NavLink bileşenleri ile akıcı bir kullanıcı deneyimi oluşturmayı öğrendim.
Dinamik Route Yönetimi: URL parametrelerini (:id) kullanarak, kullanıcının seçimine göre değişen dinamik içeriklerin nasıl render edildiğini deneyimledim.
