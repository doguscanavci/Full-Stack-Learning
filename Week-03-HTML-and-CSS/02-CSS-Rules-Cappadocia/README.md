# Task: CSS Rules - Cappadocia Project

## Task Description
The objective of this task is to build a pixel-perfect replica of a Cappadocia-themed website. Using the provided reference screenshot (design/screenshot.png), the goal is to implement a complete web page structure using HTML and CSS.

**Technical Requirements:**
- File Structure: Organize project assets within the assets/ folder and maintain clean index.html and index.css files.
- Implementation: - Construct the HTML skeleton including Header, Main Content, Image Gallery, and Footer.
- Apply CSS styles for layout, colors, and typography based on the design.

**Checklist:**
- Analyzed the reference design screenshot.
- Structured the HTML document correctly.
- Linked all images from the assets/ directory.
- Applied precise CSS properties (Colors, Fonts, Spacing, Alignments).
- Verified pixel-perfect matching with the original design.
- Tested the layout across different screen sizes.

---

## My Learning Journey & Reflection
Visual Hierarchy: Recreating a themed site taught me how to prioritize visual elements like hero images and galleries to create an engaging user experience.

Layout Precision: I utilized browser DevTools extensively to inspect element dimensions and ensure that every container aligns perfectly with the reference image.

Asset Management: Handling multiple high-quality images within a gallery section helped me understand the importance of organized folder structures and correct path referencing in HTML.

# Görev: CSS Kuralları - Kapadokya Projesi

## Görev Tanımı
Kapadokya temalı bir web sitesinin birebir kopyasını oluşturmanız isteniyor. Size verilen ekran görüntüsünü (design/screenshot.png) referans alarak, HTML ve CSS kullanarak aynı tasarımı yapmanız gerekiyor.

**Teknik Gereksinimler:**
- Dosya Yapısı: Proje görsellerini assets/ klasöründe düzenlemek; index.html ve index.css dosyalarını korumak.
- Yapılacaklar: - Header, Ana İçerik, Görsel Galerisi ve Footer bölümlerinden oluşan HTML yapısını kurmak.
- Tasarıma uygun CSS stillerini, renkleri ve tipografiyi uygulamak.

**Kontrol Listesi:**
- Tasarım görseli detaylıca incelendi.
- HTML yapısı anlamsal bütünlükle oluşturuldu.
- Tüm görseller assets/ klasöründen doğru şekilde çağrıldı.
- CSS özellikleri (Renkler, Yazı tipleri, Boşluklar, Hizalamalar) hassas şekilde uygulandı.
- Tasarımla birebir uyum sağlandı.
- Farklı ekran boyutlarında yerleşim test edildi.

## Gelişim Süreci ve Notlarım
Görsel Hiyerarşi: Temalı bir siteyi yeniden oluşturmak, kullanıcı deneyimini artırmak için ana görselleri ve galerileri nasıl önceliklendireceğimi öğretti.

Yerleşim Hassasiyeti: Eleman boyutlarını incelemek ve her kapsayıcının referans görselle tam olarak hizalanmasını sağlamak için tarayıcı DevTools (Geliştirici Araçları) özelliğini yoğun bir şekilde kullandım.

Varlık Yönetimi: Bir galeri bölümünde birden fazla yüksek kaliteli görselle çalışmak, organize klasör yapılarının ve HTML'deki doğru yol referanslarının önemini kavramamı sağladı.
