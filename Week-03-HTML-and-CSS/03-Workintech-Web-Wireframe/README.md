# Task: New Job First Task - Wireframe Implementation

## Task Description
As part of a new project proposal for an education company called "Workintech," I was tasked with creating a functional wireframe. The goal is to represent the fundamental structure and layout of the website using simple boxes and lines, following the provided design/wireframe.png reference.

**Technical Requirements:**
- Layout Construction: Build the core skeleton based on the wireframe template using index.html.
- Styling Strategy: Use index.css to define the spatial relationships between components without focusing on high-fidelity visual details.

**Checklist:**
- Analyzed the layout structure in wireframe.png.
- Followed all instructions in the HTML and CSS files.
- Placed UI elements and content blocks according to the plan.
- Ensured correct alignment and spacing using modern CSS layout techniques.
- Verified the wireframe's structural integrity across the browser.

---

## My Learning Journey & Reflection
Structural Planning: This task helped me understand that a good website starts with a solid blueprint. Focusing on "boxes and lines" allowed me to master layout logic before getting distracted by colors or images.

Layout Precision: I utilized browser DevTools to ensure that each section of the wireframe occupied the correct proportional space, creating a balanced and professional skeleton for the Workintech site.

Semantic Wireframing: Even in a simple taslak (draft), I practiced using semantic tags like nav, aside, and main to keep the foundation of the project professional from the very beginning.

---

## Tech Stack
- HTML5 (Structural Layout)
- CSS3 (Flexbox/Grid for Skeleton)
- UI/UX Wireframing Principles

### Türkçe

# Görev: Yeni İş İlk Görev - Wireframe Uygulaması

## Görev Tanımı
"Workintech" isimli eğitim firması için hazırlanan web sitesi teklif sürecinde, sitenin temel yapısını ve düzenini gösteren bir wireframe (taslak) oluşturmam istendi. design/wireframe.png dosyasındaki yerleşimi referans alarak, basit kutular ve çizgilerle kullanıcı arayüzü öğelerini planlamam hedeflendi.

**Teknik Gereksinimler:**
- Yerleşim Kurgusu: index.html kullanarak wireframe şablonuna dayalı temel iskeleti oluşturmak.
- Stil Stratejisi: Görsel detaylara odaklanmadan, bileşenler arasındaki boşluk ve konum ilişkilerini index.css ile tanımlamak.

**Kontrol Listesi:**
- wireframe.png üzerindeki yerleşim yapısı analiz edildi.
- HTML ve CSS dosyalarındaki tüm yönergeler takip edildi.
- Kullanıcı arayüzü öğeleri ve içerik blokları plana göre yerleştirildi.
- Modern CSS yerleşim teknikleri kullanılarak doğru hizalama ve boşluklandırma sağlandı.
- Tarayıcı üzerinde wireframe'in yapısal bütünlüğü doğrulandı.

---

## Gelişim Süreci ve Notlarım
Yapısal Planlama: Bu görev, iyi bir web sitesinin sağlam bir planla başladığını anlamamı sağladı. Sadece "kutular ve çizgilerle" çalışmak, renkler veya görsellerle vakit kaybetmeden yerleşim mantığını kavramama yardımcı oldu.

Yerleşim Hassasiyeti: Tarayıcı DevTools (Geliştirici Araçları) özelliğini kullanarak, wireframe'in her bölümünün doğru orantılı alanı kaplamasını sağladım ve Workintech sitesi için dengeli bir iskelet oluşturdum.

Anlamsal Taslak Oluşturma: Basit bir taslak çalışmasında bile nav, aside ve main gibi etiketleri kullanarak, projenin temelini en baştan profesyonel standartlarda tutma pratiği yaptım.

---

## Kullanılan Teknolojiler
- HTML5 (Yapısal Yerleşim)
- CSS3 (Flexbox/Grid ile İskelet Oluşturma)
- UI/UX Wireframe Prensipleri
