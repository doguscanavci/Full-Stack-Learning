# Task: Ergineer Mangal - Responsive Header

## Task Description
Your friend Onur opened a restaurant called "Ergineer Mangal," but his website was left unfinished. In this first task, I took over the project to build a professional, responsive header that works perfectly on both desktop and mobile devices.

**Design Reference:**
- https://ibb.co/album/ZzgXPp

**Requirements:**
- Implement the header structure using semantic HTML.
- Apply CSS styles to match the desktop and mobile references.
- Use background images and flexbox for precise alignment.

---

## My Learning Journey & Reflection
Real-World Problem Solving: This task felt different because it was for a "friend's business." I learned how to translate a design vision into a functional entry point for a brand.
Background Image Handling: I practiced using background-image and background-size: cover to create a hero section that looks professional on any screen.

---

## Tech Stack
- HTML5 / CSS3
- Flexbox
- Responsive UI Design
