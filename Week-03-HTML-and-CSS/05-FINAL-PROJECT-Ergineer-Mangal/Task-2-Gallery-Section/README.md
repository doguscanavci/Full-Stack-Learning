# Task: Ergineer Mangal - Interactive Gallery

## Task Description
To make the restaurant's website more engaging, we added a photo gallery showcasing the menu items. The focus was on layout balance and interactive styling.

**Design Reference: **
- https://ibb.co/album/m69V5q

**Requirements:**
- Create a visual gallery section using images from the design assets.
- Use advanced CSS selectors like :nth-child(2n+1) for a dynamic layout.
- Master the use of space-between vs. space-around for image distribution.

---

## My Learning Journey & Reflection
Advanced Alignment: Experimenting with different justify-content values helped me understand how to distribute white space effectively.
Structural Logic: Using :nth-child allowed me to apply specific styles without adding extra classes to my HTML.

---

- Advanced Selectors

### Türkçe

# Görev: Ergineer Mangal - İnteraktif Galeri

## Görev Tanımı
Restoranın web sitesini daha ilgi çekici hale getirmek için menüdeki yemekleri sergileyen bir fotoğraf galerisi ekledik. Odak noktamız yerleşim dengesi ve interaktif stillerdi.

**Tasarım Referansı:**
- https://ibb.co/album/m69V5q

**Beklenenler:**
- Tasarım varlıklarındaki görselleri kullanarak görsel bir galeri bölümü oluşturmak.
- Dinamik bir düzen oluşturmak için :nth-child(2n+1) gibi gelişmiş CSS seçicileri kullanmak.
- En iyi görsel dağılımı için space-between ve space-around kullanımı üzerinde ustalaşmak.

---

## Gelişim Süreci ve Notlarım
Gelişmiş Hizalama: Farklı justify-content değerlerini denemek, profesyonel bir galeride beyaz alanı nasıl etkili bir şekilde dağıtacağımı anlamama yardımcı oldu.
Yapısal Mantık: :nth-child kullanmak, HTML'ime fazladan class eklemeden belirli öğelere özel stiller uygulamamı sağladı.

---

## Kullanılan Teknolojiler
- CSS3 Flexbox
- İleri Seviye Seçiciler
