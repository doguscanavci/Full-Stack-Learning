# Task: Full Stack Education Site Implementation

## Task Description
This week, your manager paired you with a buddy to complete several projects. Your current project is to build a website for an institution that provides Full Stack Web Developer training. Your teammate has completed their part, preparing the semantic HTML and CSS style files.

Your responsibility is to add the texts and image links from the siteContent object in the index.js file to the relevant elements in the document. You will work exclusively in the index.js file.

**Important Notes:**
- You can view the target design at this link "https://i.ibb.co/9qfK22h/sayfa.png".
- You can find the target HTML schema in the original.html file.

**Requirement:** Navigation links must have the italic class, and the <a> tag in the footer must have the bold class.

**Reminder:** All your code must be written in the index.js file.

**Working files:** index.js

---

## My Learning Journey & Reflection
Advanced DOM Mapping: I practiced mapping a multi-layered JavaScript object to a full-page layout. This included handling nested properties for main content sections and contact information.
Style & Class Application: Beyond content injection, I practiced dynamic class management by applying specific styles (italic and bold) to navigation and footer elements as required by the design schema.

---

## Tech Stack
- JavaScript (Advanced DOM Manipulation)
- HTML5 / CSS3
