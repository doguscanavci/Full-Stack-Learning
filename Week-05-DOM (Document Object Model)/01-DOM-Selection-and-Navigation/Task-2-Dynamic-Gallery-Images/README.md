# Task: Ergineer Mangal Dynamic Gallery Images

## Task Description
Following the success and great teamwork on the previous task, the manager assigned an additional task to convert a previously static website's images into a dynamic structure. The objective is to use an array in index.js containing image links and alt texts to populate the gallery.

**Requirements:**
- Iterate through the provided array and target all images within the div with the .gallery-content class.
- Add the image links (URLs).
- Add the alt texts using the setAttribute method.
- Add the border-md class to every image using the classList.add method.

Note: The CSS file link is currently broken; it must be fixed first to ensure the layout displays correctly.

---

## My Learning Journey & Reflection
Selection & Iteration: I practiced using querySelectorAll to target specific groups of elements and synchronized them with data using a for loop.
DOM Attribute & Class Management: I gained hands-on experience with setAttribute for accessibility (alt tags) and classList.add for dynamic styling, ensuring the project meets professional standards.
