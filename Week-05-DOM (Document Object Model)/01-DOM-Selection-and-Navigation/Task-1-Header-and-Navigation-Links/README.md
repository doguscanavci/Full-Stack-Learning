### English

# Task: Dynamic Header and Navigation Links

## Task Description
Starting a new week, your manager paired you with a teammate as a buddy during the weekly team meeting. Together, you will complete various dynamic website tasks.
You decided on the following division of labor:
Your buddy will create the semantic HTML and write the CSS code for the required style rules.
You will use the JS side to add the texts and links to the page.
Your first task is to create the header section of a page. The desired design is here "https://i.ibb.co/WVz9TsY/s5d1-task1-design.jpg".
Your buddy has done their part. All you need to do is transfer the information within the siteContent object in the index.js file to the correct elements.

**Tips:**
- Use getElementById to edit the logo.
- Use getElementsByTagName to edit the page title.
- You are free to choose the method for the links.
- Note: While getElementById returns a single element, getElementsByTagName returns an array-like collection. You can access the $i^{th}$ element of arrays by its index: For example, array[i].

**Working files:** index.css, index.js

---

## My Learning Journey & Reflection
Element Selection Methods: I learned the practical differences between getElementById (returning a single element) and getElementsByTagName (returning a collection). Understanding how to iterate through or access specific indices in a collection was key.

Separation of Concerns: This "buddy" task simulated a real-world workflow where one developer handles the structure (HTML/CSS) while another handles the dynamic data injection (JS).

---

## Tech Stack
- JavaScript (DOM Manipulation)
- HTML5 / CSS3

### Türkçe

# Görev: Web Sayfalarında Header ve Navigasyon Linkleri

## Görev Tanımı
Yeni bir haftaya başlarken haftalık ekip toplantısında yöneticin seni bir takım arkadaşın ile buddy yaptı. İkiniz beraber bazı dinamik web sitesi görevlerini tamamlayacaksınız.
Aranızdaki görev bölümünü de şu şekilde oluşturmaya karar verdiniz:
Arkadaşın semantik HTML'i oluşturacak ve gereken stil kuralları için css kodlarını yazacak.
Sen de JS tarafını kullanarak sayfadaki metinleri, linkleri ekleyeceksin.
İlk göreviniz bir sayfanın header bölümünü yapmak. İstenen tasarım burada "https://i.ibb.co/WVz9TsY/s5d1-task1-design.jpg".
Arkadaşın kendi üzerine düşen tarafı yaptı. Senin tek yapman gereken index.js dosyasındaki siteContent objesi içindeki bilgileri doğru elementlere aktarmak.

**İpuçları:**
- Logoyu düzenlemek için getElementById kullan.
- Sayfa başlığını düzenlemek için getElementsByTagName kullan.
- Linkler için metod seçmekte özgürsün.İpucu: getElementById bir element dönerken, getElementsByTagName array benzeri bir collection dönüyor. Array'lerin de $i.$ elemanına index'i ile erişebilirsin: Örneğin array[i].

**Çalışma dosyaları:** index.css, index.js

---

## Gelişim Süreci ve Notlarım
Element Seçme Metotları: getElementById (tek bir element döner) ve getElementsByTagName (bir koleksiyon döner) arasındaki farkları uygulamalı olarak öğrendim. Bir koleksiyon içindeki belirli indexlere nasıl erişileceğini kavramak bu görevin kilit noktasıydı.

Görev Paylaşımı (Buddy Sistemi): Bu çalışma, bir geliştiricinin yapıyı (HTML/CSS), diğerinin ise dinamik veri aktarımını (JS) üstlendiği gerçek dünya iş akışını simüle etmemi sağladı.
