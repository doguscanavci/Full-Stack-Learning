### English

# Day Project: Cyprus Tour Website

## Task Description
While reviewing old company web pages, you encountered an interesting site about Cyprus. Since you personally enjoy Cyprus, you decided to revitalize this static page by creating 4 different challenges. Your goal is to solve these 4 challenges using only JavaScript within the index.js file.

**Requirements:**
- Implement 4 distinct challenges within the index.js file to add interactivity.
- Ensure all modifications and logic are handled purely through JavaScript (no direct HTML changes).
- Focus on transforming static elements into dynamic, interactive components.

**Working files:** index.js

---

## My Learning Journey & Reflection
Problem Solving: I approached the project by breaking it down into 4 specific challenges, which helped me practice structured coding and modular logic.
Dynamic Interaction: By working on an existing static page, I gained experience in "re-animating" content and enhancing user experience (UX) through targeted DOM events and functional programming.

---

## Tech Stack
- JavaScript (Advanced Event Handling)
- HTML5 / CSS3

### Türkçe

# Gün Projesi: Turistik Kıbrıs Gezisi

## Görev Tanımı
Çalıştığın şirketin eskiden yaptığı web sayfalarını incelerken eğlenceli bir sayfa ile karşılaştın. Özellikle Kıbrıs'ı sevdiğin için de ayrıca ilgini çekti. Fakat bir sorun var... Sayfa çok statik. Bunu nasıl hareketlendireceğini düşünürken kendine 4 farklı challenge ayarladın.

**Beklenenler:**
- index.js dosyasındaki bu 4 challenge'ı sadece index.js'de yazacağın kodlar ile çözmek.
- Statik sayfayı kullanıcı etkileşimine açık, dinamik bir yapıya dönüştürmek.
- Tüm mantıksal çözümleri ve DOM müdahalelerini saf JavaScript kullanarak gerçekleştirmek.

**Çalışma dosyaları:** index.js

---

## Gelişim Süreci ve Notlarım
Problem Çözme: Projeyi 4 ayrı challenge'a bölerek ele almak, yapılandırılmış kodlama ve modüler mantık kurma becerilerimi geliştirdi.
Dinamik Etkileşim: Mevcut bir statik sayfa üzerinde çalışarak, içerikleri "yeniden canlandırma" ve hedeflenmiş DOM eventleri ile kullanıcı deneyimini (UX) iyileştirme konusunda deneyim kazandım.

---

## Kullanılan Teknolojiler
