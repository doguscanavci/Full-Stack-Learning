# Task: IconButton Component

## Task Description
Today at work, you attended a useful training session where you learned how to use a library called FontAwesome to integrate consistent icons into your projects. To practice, you've been assigned to create a dynamic, custom-designed icon button component. The styling for the buttons has already been prepared by a UX developer; your task is to write the function that generates these buttons.

**Requirements:**
- Complete the logic inside iconButton.js to create dynamic button elements.
- Integrate FontAwesome icons within the buttons.
- Use createElement and appendChild methods to build the component structure.

**Working files:** components/iconButton/iconButton.js

---

## My Learning Journey & Reflection
Library Integration: I learned how to incorporate third-party icon libraries like FontAwesome into a DOM-based project to enhance UI consistency.
Component Logic: I practiced writing reusable functions that return DOM elements, a fundamental concept that bridges the gap between vanilla JS and modern frameworks like React.

---

- JavaScript (Component Functions)
- FontAwesome Library
