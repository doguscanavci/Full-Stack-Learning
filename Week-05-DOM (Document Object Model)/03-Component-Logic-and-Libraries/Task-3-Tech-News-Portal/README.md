# Day Project: Tech News Portal

## Task Description
In this project, you will develop a dynamic technology news portal. The project consists of two main dynamic modules that must be built using JavaScript and DOM manipulation.

1. News Section:
Display news data provided in JSON format on the screen using the DOM. Follow the specific instructions provided in the ./components/haber/haberler.js file to implement this section.

2. Menu Section:
The portal's navigation menu must also be created dynamically. Follow the instructions in the ./components/menu/menu.js file to build the dynamic menu structure.

**Working files:** components/haber/haberler.js, components/menu/menu.js

---

## My Learning Journey & Reflection
JSON Data Mapping: I practiced parsing and mapping JSON data structures to complex UI components, ensuring that news content is rendered accurately and dynamically.
Modular Component Architecture: By separating the Menu and News into distinct components, I learned how to manage different parts of a web application independently while maintaining a unified interface.

---

## Tech Stack
- JavaScript (ES6+ Modules & DOM)
- JSON Data Handling

### Türkçe

# Gün Projesi: Teknoloji Haberleri Sitesi

## Görev Tanımı
Bu projede, dinamik bir teknoloji haber portalı hazırlayacaksın. Proje, JavaScript ve DOM manipülasyonu kullanılarak oluşturulması gereken iki ana dinamik modülden oluşmaktadır.

1. Haberler Bölümü:
JSON formatında verilen haber verilerini DOM kullanarak ekranda göstermelisin. Bu bölümü uygulamak için ./components/haber/haberler.js dosyasındaki talimatları yerine getirmelisin.

2. Menü Bölümü:
Portalın navigasyon menüsü de tamamen dinamik olarak oluşturulmalıdır. Dinamik menü yapısını kurmak için ./components/menu/menu.js dosyasındaki yönergeleri takip etmelisin.

**Çalışma dosyaları:** components/haber/haberler.js, components/menu/menu.js

---

## Gelişim Süreci ve Notlarım
JSON Veri Eşleştirme: JSON veri yapılarını çözümleyip karmaşık kullanıcı arayüzü bileşenlerine eşleyerek, haber içeriklerinin doğru ve dinamik bir şekilde render edilmesini sağladım.
Modüler Bileşen Mimarisi: Menü ve Haberleri ayrı bileşenlere ayırarak, bir web uygulamasının farklı bölümlerini bağımsız olarak yönetmeyi ve aynı zamanda bütünleşik bir arayüz sunmayı öğrendim.
