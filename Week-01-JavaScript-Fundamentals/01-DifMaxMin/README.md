# Task: Max-Min Difference (School Success Research) 

## Task Description 

A new research project is being conducted at your graduated school to measure success. They want to measure the change in the difference between the highest and lowest grades in classes over time and according to the teacher. An application is needed to calculate this difference for each class separately.

**Expected:**
- Write a function named `difMaxMin`.
- The function should take class grades (array) as a parameter.
- It should return the difference between the maximum and minimum numbers in the array.

**Example:**
- `difMaxMin([1, 5, 11, 145, 51, 2]);` // Output: 144
- `difMaxMin([2, 9, -13, 14]);` // Output: 27
